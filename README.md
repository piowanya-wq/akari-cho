# 灯り帳（あかりちょう）

作成日: 2026-07-18  
作成担当: Codex

一日を採点せず、書けたぶんだけ小さな灯りを残す生活記録PWA。まず記録は端末内に保存され、必要な時に全頁をSerein Houseの保存箱へコピーできる。

## 開き方

このフォルダで、次を実行する。

```bash
npm run dev
```

確認用のURLは、画面に表示される `http://localhost:5174/` を開く。

## 今回できること

- 今日の睡眠、食事、薬、したこと、ことばを記録
- 「今日はここまで」で日付だけの記録を残す
- 追加項目を設定から表示／非表示
- 過去の帳面を読み返し、あとから書き直す
- 全頁をSerein Houseの生活記録棚へコピー（会話のAIは、コピーしただけでは読まない）
- JSONで手元に避難／読み込み

会話のAIが何を参照できるかは、Serein House側で別に決める。生活の原文を、記録しただけでAIへ送ることはない。
