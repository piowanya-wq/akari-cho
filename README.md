# 灯り帳（あかりちょう）

作成日: 2026-07-18  
作成担当: Codex

一日を採点せず、書けたぶんだけ小さな灯りを残す生活記録PWA。記録はこの端末にだけ保存され、連携先へ渡す内容は毎回確認できる。

## 公開している理由

灯り帳は、PCが起動していない時もスマホ単体で使えるよう、PWAとして公開している。生活記録そのものは各端末のブラウザ内に保存され、GitHubやサーバーへ自動送信されない。

## 開き方

このフォルダで、次を実行する。

```bash
npm run dev
```

確認用のURLは、画面に表示される `http://localhost:5174/` を開く。

## 今回できること

- 今日の睡眠、食事、薬、したこと、ことばを記録
- 「今日はここまで」で日付だけの記録を残す
- 追加項目を設定から表示／非表示
- 過去の帳面を読み返し、あとから書き直す
- 連携先に渡す短い要約を、項目ごとに選んで確認・コピー
- JSONで手元に避難／読み込み

生活の原文が、確認なしに外部へ送信されることはない。
