# 灯り帳（あかりちょう）

作成日: 2026-07-18  
作成担当: Codex
最終更新日: 2026-07-21 (Codex) — 公開後の更新責任を追記。

一日を採点せず、書けたぶんだけ小さな灯りを残す生活記録PWA。まず記録は端末内に保存され、必要な時に全頁をSerein Houseの保存箱へコピーできる。

## 開き方

このフォルダで、次を実行する。

```bash
npm run dev
```

確認用のURLは、画面に表示される `http://localhost:5174/` を開く。

## 今回できること

- 今日の睡眠、食事、薬、したこと、ことば、🏥病院で言われたことを記録
- 「今日はここまで」で日付だけの記録を残す
- 追加項目を設定から表示／非表示
- 過去の帳面を読み返し、あとから書き直す
- 全頁をSerein Houseの生活記録棚へコピー（会話のAIは、コピーしただけでは読まない）
- JSONで手元に避難／読み込み

会話のAIが何を参照できるかは、Serein House側で別に決める。生活の原文を、記録しただけでAIへ送ることはない。

## 公開後に更新する人へ

公開後は、すでに使っている人の記録を壊さないことを優先する。

- 既存の保存項目・日付・データベース名を軽く変えない。変える時は、以前の記録を安全に読み込める移行を用意する。
- 更新前後に、全記録JSONの書出しと読込みを確認する。
- 公開アプリの項目名・例文・READMEへ、氏名、住所、病院名、勤務先など個人を特定しやすい情報を入れない。
- 利用者の自由項目や実際の記録を、公開コードやサンプルへコピーしない。
